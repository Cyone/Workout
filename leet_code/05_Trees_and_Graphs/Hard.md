# Hard

### 1. 124. Binary Tree Maximum Path Sum
Given the `root` of a binary tree, return the maximum path sum of any non-empty path.

**Test Cases:**
1. `root = [1,2,3]` -> `6`
2. `root = [-10,9,20,null,null,15,7]` -> `42`
3. `root = [-3]` -> `-3`
4. `root = [2,-1]` -> `2`
5. `root = [1,-2,3]` -> `4`

### 2. 297. Serialize and Deserialize Binary Tree
Design an algorithm to serialize and deserialize a binary tree into a string.

**Test Cases:**
1. `root = [1,2,3,null,null,4,5]` -> `[1,2,3,null,null,4,5]`
2. `root = []` -> `[]`
3. `root = [1]` -> `[1]`
4. `root = [1,2]` -> `[1,2]`
5. `root = [1,null,2]` -> `[1,null,2]`
