### 1. 4. Median of Two Sorted Arrays
Given two sorted arrays `nums1` and `nums2` of size `m` and `n` respectively, return the median of the two sorted arrays.

**Test Cases:**
1. `nums1 = [1,3], nums2 = [2]` -> `2.00000`
2. `nums1 = [1,2], nums2 = [3,4]` -> `2.50000`
3. `nums1 = [0,0], nums2 = [0,0]` -> `0.00000`
4. `nums1 = [], nums2 = [1]` -> `1.00000`
5. `nums1 = [2], nums2 = []` -> `2.00000`

### 2. 51. N-Queens
The n-queens puzzle is the problem of placing `n` queens on an `n x n` chessboard such that no two queens attack each other. Return all distinct solutions.

**Test Cases:**
1. `n = 4` -> `[[".Q..","...Q","Q...","..Q."],["..Q.","Q...","...Q",".Q.."]]`
2. `n = 1` -> `[["Q"]]`
3. `n = 2` -> `[]`
4. `n = 3` -> `[]`
5. `n = 5` -> `10 solutions (just check length is 10)`
