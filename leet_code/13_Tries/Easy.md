# Easy

### 1. 720. Longest Word in Dictionary

Given an array of strings `words`, return the longest word in `words` that can be built one character at a time by other words in the array. If there is more than one possible answer, return the one with the smallest lexicographical order.

**Test Cases:**

1. `words = ["w","wo","wor","worl","world"]` -> `"world"`
2. `words = ["a","banana","app","appl","ap","apply","apple"]` -> `"apple"`
3. `words = ["m","mo","moc","mocha"]` -> `"mocha"`
4. `words = ["a"]` -> `"a"`
5. `words = ["yo","ew","fc","zrc","yodn","fcm","qm","qmo","fcmz","z","ewq","yod","ewqz","y"]` -> `"yodn"`
