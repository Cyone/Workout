# Easy

### 1. 70. Climbing Stairs

You are climbing a staircase with `n` steps. You can climb 1 or 2 steps at a time. In how many distinct ways can you climb to the top?

**Test Cases:**

1. `n = 2` -> `2`
2. `n = 3` -> `3`
3. `n = 1` -> `1`
4. `n = 5` -> `8`
5. `n = 10` -> `89`

### 2. 198. House Robber

You are a robber planning to rob houses along a street. Adjacent houses cannot be robbed on the same night. Given `nums`, return the maximum amount you can rob.

**Test Cases:**

1. `nums = [1,2,3,1]` -> `4`
2. `nums = [2,7,9,3,1]` -> `12`
3. `nums = [1]` -> `1`
4. `nums = [2,1]` -> `2`
5. `nums = [5,1,1,5]` -> `10`

### 3. 746. Min Cost Climbing Stairs

You are given an integer array `cost` where `cost[i]` is the cost of the `ith` step. Once you pay the cost, you can climb 1 or 2 steps. Return the minimum cost to reach the top.

**Test Cases:**

1. `cost = [10,15,20]` -> `15`
2. `cost = [1,100,1,1,1,100,1,1,100,1]` -> `6`
3. `cost = [0,0]` -> `0`
4. `cost = [1,2,3]` -> `2`
5. `cost = [10,15]` -> `10`

### 4. 509. Fibonacci Number

The Fibonacci number is defined as `F(n) = F(n-1) + F(n-2)`. Compute `F(n)`.

**Test Cases:**

1. `n = 2` -> `1`
2. `n = 3` -> `2`
3. `n = 4` -> `3`
4. `n = 10` -> `55`
5. `n = 0` -> `0`
